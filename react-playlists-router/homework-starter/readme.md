# React Playlists Router

Учебный проект для практики React Router.

## Функциональность

- маршрутизация с помощью react-router-dom
- динамические маршруты
- переход между страницами
- фильтрация плейлистов
- работа с search params

## Маршруты

- `/` — главная
- `/users` — список пользователей
- `/users/:userId` — информация о пользователе
- `/playlists` — список плейлистов
- `/playlists/:playlistId` — информация о плейлисте

## Используемые технологии

- React
- TypeScript
- React Router
- Vite
